# Changelog

All notable project changes should be recorded here.

## 0.2.0 - 2026-06-11

- Initial public release.
- Added local desktop recording for meeting audio and optional microphone input.
- Added CPU/int8 Faster Whisper transcription.
- Added trusted pinned model definitions for `tiny`, `base`, `small`, and
  `medium`.
- Added owner-only local file handling for settings, audio, temporary tracks,
  and transcripts on POSIX systems.
- Added PyInstaller build scripts, smoke checks, Docker Linux build support, and
  OS-specific usage/build documentation.
- Added third-party license notices and disabled public CI binary artifact
  upload until binary redistribution review is complete.
