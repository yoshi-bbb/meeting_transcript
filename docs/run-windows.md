# Windows 版 実行ファイルの使い方（エンドユーザー向け）

このドキュメントでは、Windows でビルド済みの Meeting Mojiokoshi 実行ファイル（.exe）を実行する方法を説明します。

## 入手方法
現在、GitHub Releases では一般利用者向けのビルド済み ZIP を提供していません。

- ソースから自分でビルドした `dist/MeetingMojiokoshi.exe` を使用してください。
- ビルド手順は `docs/build-windows.md` を参照してください。

## 実行手順

1. ZIP ファイルを任意のフォルダに展開します。
   - 例: `C:\Tools\MeetingMojiokoshi\` やデスクトップ上

2. 展開されたフォルダ内の `MeetingMojiokoshi.exe` をダブルクリックします。

3. 初回起動時は Windows SmartScreen やファイアウォールの警告が出る場合があります。
   - 「詳細情報」→「実行」を選択してください（信頼できるソースからダウンロードした場合）。

## System Tray（タスクトレイ）機能

アプリをバックグラウンドで動作させたい場合に便利です。

- メインウィンドウを閉じると、自動的にシステムトレイ（タスクバー右側の通知領域）に最小化されます。アプリは終了せず、録音を継続できます。
- トレイアイコンを右クリックするとメニューが表示されます：
  - 「ウィンドウを開く」：メイン画面を表示
  - 「録音開始」：録音を開始（状態に応じて有効/無効）
  - 「停止して文字起こし」：録音を停止して文字起こし（状態に応じて有効/無効）
  - 「終了」：アプリを完全に終了
- トレイアイコンは常に表示されます（ウィンドウを開いていてもOK）。
- 録音中はトレイから素早く操作可能で、デスクトップをすっきり保てます。

## 初回設定
- アプリ起動後、「出力先」フォルダを選択します（デフォルトは `ドキュメント\MeetingMojiokoshi`）。
- 「会議音声」デバイスを選択（システム音声 / loopback デバイス）。
- 「マイク」デバイスを選択（不要なら「使用しない」）。
- モデル（tiny / base など）を選択し、「モデル準備」をクリック（初回はモデルをダウンロードします）。
- 「録音開始」をクリックしてミーティングを開始。
- 終了後に「停止して文字起こし」をクリック。
- 出力先に `meeting_YYYYMMDD_HHMMSS.wav` と `.txt` が保存されます。

## 注意点
- 録音には参加者の同意を得てください。
- モデルは初回のみダウンロード。以降はローカルキャッシュを使用します。
- アプリは GUI アプリケーションです。ターミナルは不要です。

## アンインストール
- 展開したフォルダを削除するだけで完了です（設定はユーザーの AppData や Documents に残る場合があります）。

詳細なアプリの使い方は同梱の README.md またはアプリ内ヘルプを参照してください。
